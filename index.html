About Me

            Nama saya Muhammad Dzaky Taqillah. Saya biasa dipanggil Dzaky, tapi karena orang dengan nama Dzaky di kampus tempat saya kuliah ada banyak, jadi saya meminta semua teman saya di kampus untuk memanggil saya dengan panggilan Taqi. Dzaky pada nama saya memiliki makna cerdas, sedangkan Taqillah pada nama saya memiliki makna Taqwa.
